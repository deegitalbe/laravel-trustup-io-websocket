# laravel-trustup-io-websocket

## Compatibility

| Laravel | Package |
|---|---|
| 8.x / 9.x | 1.x |
| 12.x | 2.x |

## Installation

```shell
composer require deegitalbe/laravel-trustup-io-websocket
```

### .env configuration

Add these information to your `.env` file.
```dotenv
TRUSTUP_IO_WEBSOCKET_APP_SECRET=
```

You're good to go!
